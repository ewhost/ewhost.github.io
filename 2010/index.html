Outlook 2010
  1. Go to the File menu and click on Info and then Add Account.
  2. Select Manually configure server settings or additional server types and click Next.
  3. Select Internet E-Mail and then click Next
  4. Enter the following information.
    • Your Name: your name
    • Email Address: your email address
    • Account Type: IMAP
    • Incoming Mail server (IMAP): mail.mailconfig.net
    • Outgoing Mail server (SMTP): mail.mailconfig.net
    • User Name: your email address
    • Password: your email password
    • DO NOT check the Secure Password Authentication box
  5. Click on More Settings and select the Outgoing Server tab.
  6. Check My outgoing server (SMTP) requires authentication
  7. Select Use same settings as my incoming mail server.
  8. Click on the Advanced tab
  9. Change the Outgoing server (SMTP) to 587
  10. Set Use encrypted connection to SSL
  11. Click Ok. Click Next. Click Finish.
