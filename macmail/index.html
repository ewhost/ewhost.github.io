Mac Mail
  1. In Mac Mail, go to the Mail menu and click on Preferences
  2. Click the Accounts icon in the Mail Preferences window.
  3. Click + to create a new account.
  4. Select Add Other Mail Account
  5. Enter your name, email address, password and click Create:
  6. Select IMAP and for mail server enter mail.mailconfig.net
  7. Enter your email address for User Name, your email password, and click Next.
  8. For SMTP server enter mail.mailconfig.net
  9. Enter your email address for User Name, your email password, and click Next.
  10. Change the port to 587
  11. Select the Use SSL checkbox
  12. Set Authentication to Password and click Create
  13. To double-check your settings, click on the account in the list and it should closely match the following:
  14. To confirm outgoing settings, from the Outgoing Mail Server (SMTP) drop-down list select Edit SMTP List
  15. From the Advanced tab, set the port to 587
  16. Select the Use SSL checkbox
  17. Set Authentication to Password
  18. Do not select the "Allow insecure authentication" checkbox
  19. Ensure your full email address and email password are correct
  20. Click OK when finished
